## 3. WP への貼り付け範囲

各HTMLファイルから、**`<body>` 直下のコンテンツのみ**を WP の固定ページ本文（コードブロック／カスタムHTML）に貼り付けてください。

具体的には：

| 範囲 | 内容 | WP側で出力済か |
|---|---|---|
| `<!doctype html>` 〜 `</head>` | head全体 | ✅ WP出力（**貼らない**） |
| `<body class="...">` | bodyタグ | ✅ WP出力（**貼らない**） |
| `<div class="scroll-progress">` から `</a>` 最後（`floating-cta`まで） | **本文** | ❌ ここをコピペ |
| `<script src=".../main.js" defer></script>` | JS | ✅ WPテーマで読込済（**貼らない**） |
| `</body></html>` | 閉じタグ | ✅ WP出力（**貼らない**） |

貼り付け開始：`<div class="scroll-progress" aria-hidden="true"></div>`  
貼り付け終了：`<a class="floating-cta btn btn--line btn--block" ...>LINEで...</a>`

※不要であれば、各HTMLの `<head>` 〜 `</head>` と `<body>` `</body></html>` を削除した状態の貼り付け用 partial を、別途生成可能です。ご指示ください。
